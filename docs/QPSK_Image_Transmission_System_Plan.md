## 1. 系统概述

### 1.1 项目目标
设计并实现一个完整的医学图像(DICOM)无线传输系统，基于QPSK调制和12.5MHz中频，通过毫米波信道实现PC-to-PC的单帧图像传输。

### 1.2 系统规格

| 参数 | TX端(FPGA+DAC) | 信道 | RX端(ADC+FPGA) |
|------|----------------|------|----------------|
| **硬件** | 3PD5651E | - | 3PA1030 |
| **采样率** | **125 MSPS** | - | **50 MSPS** |
| 中频频率 | 12.5 MHz | 外部混频至毫米波 | 12.5 MHz IF输入 |
| 调制方式 | QPSK (Gray编码) | - | QPSK解调 |
| **符号率** | **6.25 Msymbol/s** | - | **6.25 Msymbol/s** |
| 插值/抽取 | 20x插值 (125/6.25) | - | 8x抽取 (50/6.25) |
| 数据接口 | UDP网口 | 无线 | UDP网口 |
| 应用场景 | 单帧DICOM图像传输 | - | 单帧DICOM图像接收 |

**注意**: TX和RX采样率不同(125MSPS vs 50MSPS)，但符号率保持一致(6.25M)。这是硬件决定的架构约束。

## 3. 关键参数计算

### 3.1 时钟与采样率规划

| 时钟域 | 频率 | 用途 | 来源 |
|--------|------|------|------|
| F_ref | 50 MHz | 参考时钟 | 外部晶振 |
| F_adc_dac | 50 MHz | ADC/DAC采样时钟 | PLL生成 |
| F_gmii | 125 MHz | 以太网GMII时钟 | PLL生成 |
| F_sym | 6.25 MHz | 符号时钟 | PLL生成 (F_gmii/20) |

**TX端参数 (125MSPS系统):**
```
符号率 Rs = 6.25 Msymbol/s
采样率 Fs_tx = 125 MSPS
插值比 = Fs_tx/Rs = 20 samples/symbol

RRC滤波器参数:
- 滚降系数 α = 0.35
- 信号带宽 BW = Rs*(1+α)/2 = 4.21875 MHz
- 占用带宽 OBW = Rs*(1+α) = 8.4375 MHz
- IF频率 F_if = 12.5 MHz (满足 F_if > BW)
```

**RX端参数 (50MSPS系统):**
```
符号率 Rs = 6.25 Msymbol/s (与TX相同)
采样率 Fs_rx = 50 MSPS
抽取比 = Fs_rx/Rs = 8 samples/symbol

DDC架构:
ADC(50MSPS) → 混频(12.5MHz) → CIC(8x抽取) → 匹配滤波 → 6.25MHz符号
```

### 3.3 数据速率计算

| 参数 | 数值 | 说明 |
|------|------|------|
| 原始比特率 | 12.5 Mbps | 2 bits/symbol × 6.25 Msymbol/s |
| 有效数据率 | ~10 Mbps | 考虑帧头、CRC、协议开销 (~80%效率) |
| 单帧DICOM大小 | 典型512KB - 2MB | 取决于图像分辨率 |
| 单帧传输时间 | 0.4 - 1.6秒 | @10Mbps有效速率 |
| UDP包大小 | 1472字节 | 以太网MTU限制 |
| 每帧包数 | 350 - 1400包 | 2MB/1472字节 |

### 3.4 缓冲区大小计算

```
TX端FIFO设计:
- 写时钟: 125 MHz (GMII)
- 读时钟: 6.25 MHz (符号时钟)
- 写速率: 125M × 8bit = 1 Gbps (突发)
- 读速率: 6.25M × 8bit = 50 Mbps (连续)
- 所需FIFO深度: 至少容纳1个UDP包 = 1472字节
- 设计值: 2048字节 (2KB) 或 4096字节 (4KB)

RX端FIFO设计:
- 写时钟: 6.25 MHz (符号时钟)
- 读时钟: 125 MHz (GMII)
- 写速率: 6.25M × 8bit = 50 Mbps
- 读速率: 125M × 8bit = 1 Gbps (突发)
- 所需FIFO深度: 同上, 2048-4096字节
```

**成功标准**:

## 8. 附录

### 8.1 关键公式汇总

**QPSK调制:**
```
s(t) = I(t)·cos(2πf_ct) - Q(t)·sin(2πf_ct)

I,Q ∈ {+A/√2, -A/√2}  (Gray编码)
```

**RRC滤波器频率响应:**
```
H(f) = {
    1,                                    |f| ≤ (1-α)/(2T)
    cos[(πT/2α)(|f| - (1-α)/(2T))],     (1-α)/(2T) < |f| ≤ (1+α)/(2T)
    0,                                    |f| > (1+α)/(2T)
}
```

**Gardner定时误差检测:**
```
e(τ) = y(nT_s + τ) · [y((n+1/2)T_s + τ) - y((n-1/2)T_s + τ)]
```

**Costas环相位检测:**
```
e_φ = I·Q̂ - Q·Î ≈ sin(2Δφ) ≈ 2Δφ  (小误差近似)
```
